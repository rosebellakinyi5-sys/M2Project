# SPA - Wordly Dictionary

## Description

Worldly is a single-page dictionary application that allows one to search for any English phrases or words used to get their meaning, synonyms, pronunciation, examples as well as audio. All these are done without the page reloading.

Plain HTML,CSS and JavaScript has been used to come up with this application.

## Features

*Word Search; Instant Results.
*Definitions; Organized by parts of speech (noun, verb, etc.)
*Pronunciation/ Phonetics.
*Audio Playback.
*Example sentences (shown when available).
*Synonyms.
*Source link of the original definition entry.
*Favorite Section; One can add,remove and revisit even after refreshing the page.
*Clear error handling for words not found,empty searches,network issues.

## Roles
-HTML
-CSS
-JavaScript
-Free Dictionary API-(https://api.dictionaryapi.dev/api/v2/entries/en/)
-Web Storage API- 'localStorage

## How to use

1.Enter a word in the search box.
2.Click the "Search" button.
3.View the results displayed below the search bar.
4.Click "Play Audio' TO HEAR PRONUNCIATION.
5.Click "Add to Favorites" to save the word "Remove" to take out the word from favorites.


## Limitations
-Some words lack examples, phonetics, audio or more than one meaning.
-Favorites are stored locally in browser; Local Storage
-English words only are suppored by this application.
